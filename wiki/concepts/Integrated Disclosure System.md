# Integrated Disclosure System

The **integrated disclosure system** (created **1982**) is the architecture that unified SEC disclosure: one set of content rules ([[Regulation S-K]] for narrative, [[Regulation S-X]] for financials) feeding standardized forms (the [[SEC Filing Types|S-1, 10-K, 10-Q]], etc.). (Source: [[CFO Bridge - Regulation S-X vs S-K]])

---

## The problem it solved

Before 1982, separate disclosure regimes applied to **Securities Act** registration statements and **Exchange Act** periodic reports, producing overlapping, duplicative requirements. The integrated system made the same building blocks (S-K + S-X) populate both. (Source: [[CFO Bridge - Regulation S-X vs S-K]])

## How it fits together

- **[[Regulation S-K]]** -> narrative content (business, risk factors, MD&A, exec comp).
- **[[Regulation S-X]]** -> financial-statement form and content.
- **Forms** (S-1, 10-K, 10-Q, 8-K) -> the containers that assemble S-K + S-X items for a given purpose.
- This is why the same MD&A and audited financials appear in both a registration statement and an annual report.
